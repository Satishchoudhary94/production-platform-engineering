# Production Platform Engineering

> An enterprise-grade Internal Developer Platform (IDP) built using modern Platform Engineering principles.

---

## Overview

This repository contains the complete source code for a production-style cloud platform designed to run a microservices-based e-commerce application.

The objective of this project is to simulate how a real Platform Engineering team designs, builds, secures, deploys, and operates cloud-native infrastructure in a modern technology company.

Unlike tutorial projects, this repository focuses on production engineering practices including Infrastructure as Code, Kubernetes, GitOps, Observability, Security, CI/CD, and Site Reliability Engineering.

The application workload used throughout the project is Google's Online Boutique microservices application, while every platform component surrounding it is designed and implemented from scratch.

---

## Project Goals

- Build a reusable cloud platform
- Practice Platform Engineering
- Learn production Kubernetes
- Implement GitOps using ArgoCD
- Deploy infrastructure using Terraform
- Build production CI/CD pipelines
- Implement Observability
- Apply DevSecOps practices
- Follow SRE principles
- Migrate from Local AWS (Floci) to AWS
